# React Hooks — Pomodoro Timer 

Se construye un temporizador Pomodoro progresivo en tres niveles de complejidad.

## Video explicativo

🎥 [Ver video explicativo](https://youtu.be/Q1R0gcB_7ik)

## Hooks utilizados

- `useState`: manejo de tiempo, modo, estado de ejecución, historial y configuración
- `useEffect`: intervalo del timer, cambio automático de modo, sincronización de configuración y reproducción de sonido

## Tecnologías

- React (componentes funcionales)
- JavaScript ES6+
- CSS-in-JS (estilos en línea)
